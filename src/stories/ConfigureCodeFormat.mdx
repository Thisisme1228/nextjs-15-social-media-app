### TailwindCSS

```
1.npm i tailwindcss

2.vscode installs 'Tailwind CSS IntelliSense' plugin and settings:

//to tell VS Code to always open .css files in Tailwind CSS mode:
"files.associations": {
  "*.css": "tailwindcss"
}

//to make VS Code trigger completions when editing "string" content
"editor.quickSuggestions": {
  "strings": "on"
}
```

### Prettier

``` 
1.create a file ConfigureCodeFormat.mdx in the root folder and then write code:
module.exports = {
  plugins: ["prettier-plugin-tailwindcss"]
}

2.then install Prettier extension.

//To ensure that this extension is used over other extensions you may have installed
Editor: Default Formatter-----> prettier-Code Formatter

3..eslintrc.json
add item: "prettier"
```
